## 📊 数据字段说明

```typescript
{
  id: number;              // 唯一ID
  date: string;            // 日期 YYYY-MM-DD
  projectGroup?: string;   // 项目组（项目组A/B/C/Enhance）
  project: string;         // 项目名称
  phase: string;           // 工程阶段（'-' 表示无）
  hours: number;           // 工时
  startTime: string;       // 开始时间 HH:mm
  endTime: string;         // 结束时间 HH:mm
  description: string;     // 作业描述
  workType: '通常' | '残業' | '休日';  // 作业时间类型
  workCategory: 'normal' | 'jira' | 'management';  // 作业分类
  subType?: string;        // 子类型（保守作业专用）
  jiraNumber?: string;     // JIRA号码或主题（JIRA/チャット专用）
  modifiedAt?: string | null;  // 修改时间
}
```
